# Deepfake Face Detection

Проект по бинарной классификации изображений лиц: **настоящее** или **сгенерированное**. Реализован в рамках соревнования на Kaggle (ML Intensive, Яндекс Академия, весна 2026).

**Лучший результат на Kaggle: `0.956` (F1-score)**

---

## Задача

Дано 50 000 изображений лиц размером 256×256. Нужно определить, является ли лицо реальным или сгенерированным нейросетью (StyleGAN). Метрика — F1-score.

Особенность датасета: **сильный дисбаланс классов** — ~83% реальных и ~17% фейковых

---

## Подход

### Препроцессинг и аугментации

- **Медианный фильтр** (`cv2.medianBlur`) для удаления точечного шума перед подачей в модель
- Аугментации при обучении: горизонтальный флип, поворот, color jitter, случайный grayscale, gaussian blur
- Дисбаланс классов учитывался через `WeightedRandomSampler` и `pos_weight` в функции потерь (`BCEWithLogitsLoss`)

### Архитектуры (все написаны с нуля)

| Модель | Описание | Val F1 |
|---|---|---|
| **ResNet-34** | Baseline с residual-блоками | 0.874 |
| **ResNet-34 + FFT** | Добавлена частотная ветка (FFT) | 0.887 |
| **ResNet-34 + SE + FFT** | SE-блоки (channel attention) внутри каждого residual-блока | **0.935** |
| **ResNet-50 + SE + FFT** | Более глубокая сеть с Bottleneck-блоками | 0.922 |

**Ключевые идеи:**

- **FFT-ветка** — StyleGAN оставляет характерные артефакты в частотном домене. Параллельная ветка смотрит на `log(|FFT|)` изображения и конкатенируется с пространственными признаками
- **SE-блоки (Squeeze-and-Excitation)** — channel attention, который позволяет модели самой расставлять веса по каналам признаков

### Обучение

- Оптимизатор: `AdamW`, lr = `1e-3` → дообучение с `1e-4`
- Scheduler: `CosineAnnealingLR`
- ResNet-50 дообучался дополнительно 15 эпох после основных 30

### Постобработка

- **Подбор порога**: вместо дефолтного 0.5 найден оптимальный порог **0.69** по F1 на валидации
- **TTA (Test Time Augmentation)**: каждое тестовое изображение прогонялось 6 раз с разными аугментациями (флип, поворот, color jitter, их комбинации), результаты усреднялись
- **Ансамбль**: взвешенное усреднение двух лучших моделей — 60% ResNet-34+SE+FFT и 40% ResNet-50+SE+FFT

---

## Результаты

```
Ансамбль (ResNet-34+SE+FFT × 60% + ResNet-50+SE+FFT × 40%) + TTA×6 + threshold=0.69
→ Kaggle F1: 0.95579
```
